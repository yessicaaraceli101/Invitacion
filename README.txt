# Invitación Web (Bootstrap 5)

## Abrir
1) Descomprime el ZIP
2) Abre `index.html` (doble clic)

## Personalizar rápido
En `assets/js/app.js`:
- `eventISO` -> fecha/hora del evento
- `whatsappNumber` -> tu número (sin +)
- `eventTitle`, `eventLocation`, `googleMapsLink`

En `index.html`:
- Nombres, hora, salón, textos
- Iframe de Google Maps (opcional)

## Fotos
Reemplaza:
- `assets/img/galeria-1.jpg` ... `galeria-4.jpg`

## Publicar gratis
- Netlify (drag & drop)
- GitHub Pages
